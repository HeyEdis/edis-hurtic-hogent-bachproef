\chapter{\IfLanguageName{dutch}{Inleiding}{Introduction}}%
\label{ch:inleiding}

Binnen de organisatie van Talentguide is er een functionaliteit ontwikkeld voor hun client mtech+ om via WhatsApp arbeiders te onboarden. Dit proces bevat drie stappen:

\begin{enumerate}
    \item Vraag voor de gebruiker zijn CV
    \item Vraag voor de gebruiker zijn rol
    \item Vraag voor de gebruiker zijn functie
\end{enumerate}

Vervolgens is uit deze ongestructureerde bron, data geëxtraheerd met behulp van kunstmatige intelligentie. Die met deze data een competentieprofiel opmaakt van de gebruiker. Hiermee kan de human resource (HR) afdeling besluiten vormen om een efficiënte en doeltreffende personeelsplanning te ontwikkelen. Ook biedt het Talentguide Software-As-a-Service (SaaS) platform opleidingen aan die de werkpunten van een werknemer kunnen verbeteren zodat die beter gestuurd wordt in zijn carrière.
\bigstkip
\\
Hierbij is gevalideerde input cruciaal zodat de werknemer een doeltreffende richting voor zijn carrière kan plotten. In deze bachelorproef is onderzocht hoe de opzet van een terugkoppelingsproces kan ontwikkelend worden om de verkregen data te valideren. Zodat kwaliteitsvolle competentieprofielen gegarandeerd zijn.

Het uiteindelijke doel is om een technisch procesverslag te hebben waaruit de kritieke trigger kan worden benoemd die aanzet tot valideren, een startpunt voor het volledig human-in-the-loop proces en een functioneel prototype dat wijzigingen bijhoudt, die later door de data scientist worden gebruikt om het artificiële intelligente (AI) model te trainen.
